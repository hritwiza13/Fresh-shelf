# 🍂 FreshShelf Dashboard

A responsive, smart kitchen dashboard built with **React** and **Vite** that optimizes grocery cycles, tracks item freshness via real-time date arithmetic, and dynamically ranks shelf inventory by expiration urgency.

---

## 🚀 Features

* **Real-time Expiry Tracker:** Automatically flags items expiring within 3 days with a `⚠️ EXPIRING SOON` badge.
* **Dynamic Inventory Logs:** Interactive form to add items with specific storage types (Fridge vs. Pantry).
* **Distinct Visual Layouts:** Structural grid borders dynamically color-coded based on storage zone and freshness status.
* **Optimized Dev Build:** Powered by a lean React state machine and lightning-fast Vite compilation.

---

## 🛠️ Tech Stack

* **Frontend Library:** React.js (Hooks & Functional Components)
* **Build Tool:** Vite (Optimized for HMR)
* **Styling:** Declarative inline CSS & Semantic UI Layouts
